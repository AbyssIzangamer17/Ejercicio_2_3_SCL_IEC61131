# 🛡️ Ejercicio 2.3: SCL Validator & Generator
### Gestión Avanzada de Lógica IEC 61131-3 (Siemens TIA Portal Style)

Plataforma inteligente para la generación y validación de código SCL (Structured Control Language), diseñada con una estética moderna y funcional.

## 🌐 GitHub Pages Live Demo
Puedes acceder a la herramienta web directamente aquí:
👉 **[Acceder al Validador SCL](https://AbyssIzangamer17.github.io/Ejercicio_2_3_SCL_IEC61131/)**

## ✨ Características
- **Generación de Código**: Bloques funcionales preconfigurados para motores, válvulas y escalado analógico.
- **Validación Estructural**: Comprobación instantánea de sintaxis estándar.
- **Interfaz Premium**: Diseño basado en Glassmorphism para una experiencia de usuario superior.

## 🛠️ Stack Tecnológico
- **Frontend**: HTML5, Vanilla CSS (Outfit Font), JavaScript.
- **Backend**: Node.js & Express (para registro de auditoría local).
